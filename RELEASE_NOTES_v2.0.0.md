# Zapret2 GUI - Версия 2.0.0 (Крупное обновление)

##  Что нового в версии 2.0.0

Это мажорное обновление, полностью перерабатывающее «сердце» программы. Старый 32-битный движок заменён на актуальную версию от автора zapret (`bol-van`).

###  Полностью обновлённое ядро
- **Переход на 64-бит (x64)**: Движок `winws2.exe` обновлен с устаревшего 32-битного PE32 до актуального 64-битного PE32+.
  - Улучшена безопасность (полноценный ASLR).
  - Улучшена производительность благодаря нативным 64-битным вычислениям (LuaJIT).
- **Обновление движка до v1.0.1 (COMPAT_VER=6)**:
  - Исправлены критические баги оригинального zapret (вкл. баг `wrong ipv6 dissection`).
  - Убраны устаревшие Lua-скрипты `COMPAT_VER=5`, добавлена поддержка новейших `LUA_COMPAT_VER_REQUIRED=6` стратегий.
- **Обновление драйвера WinDivert**: Библиотеки перехвата пакетов и `cygwin1.dll` обновлены и оптимизированы.
- **Новые фильтры**: Добавлен перехват DHT-трафика (BitTorrent), улучшен перехват WireGuard.

###  Переработка пресетов (Стратегий обхода)
Мы удалили 22 старых дублирующихся пресета (от peria/Flowseal), которые использовали устаревшие и нестандартные вызовы. Вместо них создано **6 современных, сверхэффективных базовых стратегий**, написанных с нуля под синтаксис `COMPAT_VER=6`:
1. `01_General.txt` — Эталонный пресет (fake+multidisorder).
2. `02_ALT.txt` — fake+fakedsplit с tcp_ts fooling.
3. `03_ALT11.txt` — Самый популярный fake+multisplit.
4. `04_FAKE_TLS_AUTO.txt` — Автоматическая TLS-модификация (rnd,dupsid,sni).
5. `05_FAKE_TLS_AUTO_ALT.txt` — Альтернативный TLS AUTO + tcp_seq fooling.
6. `06_SIMPLE_FAKE.txt` — Простейший fake для провайдеров со слабым DPI.

Удалены "костыльные" кастомные скрипты (вроде `custom_funcs.lua`). Теперь программа использует только официальный, безопасный и сверхбыстрый код `zapret-antidpi.lua`.

---
**Важно:** Из-за перехода на 64-битное ядро, эта версия больше **не поддерживает** 32-битные системы (x86). Поддерживается Windows 8/10/11 x64.
