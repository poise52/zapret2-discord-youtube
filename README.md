**Чистая сборка для удобного обхода ограничений YouTube и Discord в России.**  
Сборка построена на базе следующего поколения движка **Zapret2** от bol-van с поддержкой синтаксиса `--lua-desync`.

Оригинальный разработчик zapret: [bol-van](https://github.com/bol-van) | Поддержать разработчика можно [здесь](https://github.com/bol-van/zapret?tab=readme-ov-file#%D0%BF%D0%BE%D0%B4%D0%B4%D0%B5%D1%80%D0%B6%D0%B0%D1%82%D1%8C-%D1%80%D0%B0%D0%B7%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D1%87%D0%B8%D0%BA%D0%B0)

> [!WARNING]
> ### 🛡️ АНТИВИРУСЫ И WinDivert
> Драйвер перехвата пакетов **WinDivert** (`WinDivert64.sys`), необходимый для работы zapret, может вызывать ложные срабатывания некоторых антивирусов. Это стандартное поведение для низкоуровневых сетевых утилит.  
> Если антивирус заблокировал запуск:
> 1. Восстановите файлы из карантина.
> 2. Добавьте папку со сборкой в исключения антивируса.

> [!IMPORTANT]
> Все исполняемые файлы в папке [`exe/`](./exe) и двоичные блобы в [`bin/`](./bin) взяты непосредственно из официального репозитория [bol-van/zapret](https://github.com/bol-van/zapret) и проверены по контрольным суммам SHA256.

## ⚙️ Использование

### 1. Включение Secure DNS (Обязательно)
Чтобы провайдер не перехватывал ваши DNS-запросы, включите DNS поверх HTTPS (DoH):
* **В Chrome/Yandex**: Настройки -> Конфиденциальность и безопасность -> Использовать безопасный DNS -> Выберите провайдера (например, Google или Cloudflare).
* **В Firefox**: Настройки -> Приватность и защита -> DNS через HTTPS -> Максимальная защита -> Выберите провайдера.
* **В Windows 11**: Настройки -> Сеть и Интернет -> Свойства вашего подключения -> Назначение DNS-сервера (изменить на ручное, включить IPv4/IPv6 DoH).
* **В Keenetic**: Если у вас роутер Keenetic, включите опцию «Транзит запросов» в настройках DNS.

### 2. Установка
1. Скачайте ZIP-архив со [страницы последних релизов](https://github.com/poise52/zapret2-discord-youtube/releases).
2. Зайдите в свойства скачанного архива и нажмите галочку **«Разблокировать»** в правом нижнем углу (если она есть), после чего нажмите Применить.
3. Распакуйте архив в любую папку **в корне диска без пробелов и кириллицы** в названии пути (например: `C:\zapret2`).

### 3. Быстрый подбор рабочей стратегии
Провайдеры блокируют трафик по-разному. В сборку встроено **22 готовых пресета**. Чтобы найти рабочий:
1. Запустите файл `service.bat` **от имени Администратора**.
2. Введите пункт **`9` (Тестирование пресетов)**.
3. Скрипт по очереди запустит тесты для популярных ресурсов на всех 22 стратегиях. Запомните номер стратегии, показавшей наилучшие результаты (где обход успешен).
4. Выберите пункт **`1` (Выбрать пресет)** и активируйте запомненный номер стратегии.
5. Выберите пункт **`2` (Установить автозапуск)**, чтобы настроить скрытый фоновый запуск обхода при старте системы через Планировщик задач Windows.

## ℹ️ Описание файлов и скриптов

* [**`service.bat`**](./service.bat) — Главный пульт управления сборкой (запускать от Администратора):
  * **`1` (Выбрать пресет)** — Выбор активной стратегии из 22 доступных пресетов в папке `presets/`.
  * **`2` (Установить автозапуск)** — Установка выбранного пресета в автозапуск через Планировщик задач Windows (запускается скрытно при входе в систему).
  * **`3` (Удалить автозапуск)** — Удаление задачи автозапуска из Планировщика.
  * **`4` (Статус службы)** — Проверка работы фоновых процессов и драйвера.
  * **`5`, `6`, `7` (Запуск / Останов / Перезапуск)** — Ручное управление фоновым процессом winws2.
  * **`8` (Диагностика)** — Проверка системы на наличие конфликтующих программ (GoodbyeDPI, старые службы zapret).
  * **`9` (Тестирование пресетов)** — Автоматизированный подбор работающей стратегии под вашего провайдера.
  * **`10` (Очистить кэш Discord)** — Быстрое удаление кэша Discord (помогает, если Discord завис на подключении).
  * **`11` (Включить TCP timestamps)** — Включение системных меток времени TCP для обхода DPI.
* [**`zapret-console.bat`**](./zapret-console.bat) — Облегченный интерактивный консольный скрипт для ручного тестирования пресетов без установки в систему.

## ☑️ Добавление собственных сайтов

Списки доменов для обхода находятся в папке `lists/`. Вы можете редактировать их в блокноте:
* **`list-general-user.txt`** — Добавьте сюда домены сайтов, которые хотите пустить через обход (например, `example.com`). Поддомены учитываются автоматически.
* **`list-exclude-user.txt`** — Добавьте сюда домены, которые вы хотите пустить в обход правил zapret (напрямую).

*После внесения изменений перезапустите процесс обхода через `service.bat` (пункт 7).*
